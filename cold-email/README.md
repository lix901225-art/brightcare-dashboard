## Email Infrastructure Setup (REQUIRED before sending)

### Current DNS Status (brightcareos.com)
- A record: ✅ (76.76.21.21 → Vercel)
- DMARC: ✅ (p=none, needs upgrade)
- MX record: ❌ MISSING
- SPF record: ❌ MISSING
- DKIM record: ❌ MISSING

### Step 1: Choose email provider

**Recommended: Google Workspace ($7.20 CAD/user/month)**
- Professional email (hello@brightcareos.com)
- Best deliverability
- Easy DKIM/SPF setup

**Alternative: Zoho Mail (Free for 1 user)**
- Free tier available
- Good deliverability
- Slightly more setup

### Step 2: Add DNS records in SiteGround

After setting up Google Workspace, add these in SiteGround DNS Zone Editor:

**MX Records:**
| Type | Name | Value | Priority | TTL |
|------|------|-------|----------|-----|
| MX | brightcareos.com | ASPMX.L.GOOGLE.COM | 1 | 3600 |
| MX | brightcareos.com | ALT1.ASPMX.L.GOOGLE.COM | 5 | 3600 |
| MX | brightcareos.com | ALT2.ASPMX.L.GOOGLE.COM | 5 | 3600 |

**SPF Record:**
| Type | Name | Value | TTL |
|------|------|-------|-----|
| TXT | brightcareos.com | v=spf1 include:_spf.google.com ~all | 3600 |

**DKIM Record:**
(Google Workspace will generate this — it's unique to your account)
| Type | Name | Value | TTL |
|------|------|-------|-----|
| TXT | google._domainkey.brightcareos.com | (generated by Google) | 3600 |

### Step 3: Upgrade DMARC

Change existing DMARC record from:
```
v=DMARC1; p=none; aspf=r; adkim=r;
```
To:
```
v=DMARC1; p=quarantine; aspf=r; adkim=r; rua=mailto:dmarc@brightcareos.com;
```

### Step 4: Warm up the domain (CRITICAL)

New email domains have poor reputation. You MUST warm up before cold sending:

- **Week 1-2:** Send 5-10 real emails/day to known contacts
- **Week 3-4:** Increase to 20-30/day
- **Week 5-6:** Increase to 50/day
- **After 6 weeks:** Can start cold outreach at 50-100/day

**Tools for automated warmup:**
- Instantly.ai (built-in warmup)
- Warmup Inbox
- Lemwarm

### Step 5: Protect primary domain

**Recommended:** Use a separate sending domain for cold email:
- Primary: brightcareos.com (for website + inbound)
- Sending: mail-brightcareos.com or try-brightcareos.com (for cold outreach)

This protects brightcareos.com reputation if cold emails get spam complaints.

```

  ├── Task 1: Enrich leads (scrape websites for emails)
  ├── Task 2: Send cold emails (via SMTP)
  ├── Task 3: Monitor inbox for replies
```
