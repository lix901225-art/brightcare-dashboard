# BrightCare OS — Staff Guide

## Adding a Child

1. Go to **Children** in the sidebar
2. Click **Add child**
3. Fill in: Full name, Date of birth, Gender, Start date
4. Select a Room
5. Click **Create**
6. After creation, edit the profile to add medical notes, allergies, and emergency info

## Inviting a Parent

1. Go to **Staff & Users** in the sidebar
2. Click **Invite by email**
3. Enter the parent's email, name, and select role (usually "Parent")
4. Click **Send invite**
5. Copy the invite link and share it with the parent
6. When the parent signs in via Auth0 with the same email, their account is automatically linked

## Creating a Daily Report

1. Go to **Daily Reports** in the sidebar
2. Click **Create report**
3. Select the child and date
4. Fill in: Meals, Naps, Mood, Activities, Bathroom, Notes
5. Optionally add photos
6. Click **Create report**
7. The parent will be automatically notified

### Batch Reports
- Click **Batch** to create reports for multiple children at once
- All children receive the same base content; add individual notes per child

## Recording an Incident

1. Go to **Records** > **Incidents** tab (or the standalone Incidents page)
2. Click **New incident**
3. Select child, severity, and type
4. Fill in description and actions taken
5. Complete BC compliance fields: witnesses, body location, parent notification
6. For formal VCH reporting, expand "Vancouver Coastal Health fields" and select categories
7. Click **Submit report**
8. The parent will be automatically notified
9. Use **VCH PDF** button to export for health authority submission
10. Use **Lock** to finalize (cannot be modified after locking)

## Generating BC Funding Reports

1. Go to **Reports** > **Funding** in the sidebar
2. Click **Create report**
3. Select report type (CCFRI, $10/Day, etc.) and month
4. The system auto-populates enrolled children and attendance days
5. Review and submit
6. Click **Export PDF** for submission to the Ministry
7. Submission deadline: **15th of each month**

## Staff Scheduling

1. Go to **Staff & Users** > click **Schedule** button
2. Use week navigation to select the target week
3. Click **Add shift** to assign a staff member to a day
4. Select shift type: Morning (07:30–12:30), Afternoon (12:30–17:30), Full day, or Off
5. The bottom row shows daily on-duty staff count

## Medical Records

1. Go to **Records** > **Medical** tab
2. **Care Plans**: Click "Add plan" to create asthma/allergy/diabetes plans
3. **Medications**: Click "Record" to log medication administration
4. All medication requires written parent authorisation per BC licensing
